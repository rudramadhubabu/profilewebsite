Below is a ULTRA PORTFOLIO CASE STUDY LAYOUT used by many Senior / Staff Product Designers when presenting complex enterprise redesign projects.
This structure is optimized for portfolio websites or long-scroll case study pages and works well with the revamp + heuristic analysis work you already created.

It is organized as clear visual sections, each designed for ~50% images / 50% text, with hero visuals and UI artifacts. You can implement each section as a Figma frame (1440px width) and later export it to your portfolio website.

Act as a senior enterprise product designer creating a complete high-fidelity UI prototype for a retail financial voucher management system.

The system is used by store employees to record incoming payments related to building damage incidents.

The UI should modernize the legacy voucher interface while preserving the same visual identity.

Design must follow these characteristics:

Dark application background
Centered white card layout
Minimal enterprise interface
Red accent actions
Compact input controls
Balanced whitespace
Clean typography

Avoid oversized buttons or oversized UI elements.

KEY NAVIGATION FEATURES:
═══════════════════════

1. HOMEPAGE SECTIONS (Anchor Links):
   • #work     → Case Studies Section
   • #about    → About Me Section  
   • #process  → Design Process Section
   • #contact  → Contact Section

2. PROJECT CARD NAVIGATION (JavaScript):
   • Click on any project card → Navigate to case study page
   • Handled by: js/main.js → initProjectCards()

3. CASE STUDY NAVIGATION:
   • All case study pages have "Back to Portfolio" button
   • Returns to: ../index.html#work (Homepage Case Studies section)

4. MOBILE MENU:
   • Hamburger menu on mobile devices
   • Same navigation links as desktop
   • Auto-closes after clicking a link

5. SMOOTH SCROLLING:
   • All anchor links scroll smoothly
   • Handled by: js/main.js → initSmoothScroll()
